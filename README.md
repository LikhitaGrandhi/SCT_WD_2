# SkillCraft Technology - Task 02

## Stopwatch Web Application

A responsive and interactive stopwatch web application developed using HTML, CSS, and JavaScript. The application allows users to accurately measure time intervals and record lap times through an intuitive user interface.

### Features

* Start, Pause, and Reset functionality
* Real-time stopwatch display
* Lap time recording and tracking
* Responsive design for different screen sizes
* User-friendly interface

### Technologies Used

* HTML
* CSS
* JavaScript

### Project Structure

```
TASK-02-STOPWATCH/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

### How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.

### Learning Outcomes

* DOM Manipulation
* Event Handling in JavaScript
* Time Functions and Intervals
* Responsive Web Design
* Front-End Development Fundamentals
